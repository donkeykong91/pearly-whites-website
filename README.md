# Pearly Whites Website

A Next.js portfolio site for a dental hygienist. The app introduces professional
background, certifications, and contact details.

## Stack

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- Radix UI (accordion/icons)
- ESLint + Prettier

## Run Locally

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev`: start development server
- `npm run build`: create production build
- `npm run start`: run production server
- `npm run lint`: run ESLint checks

## Project Structure

- `src/app`: route groups, pages, and layouts
- `src/components/ui`: reusable UI components
- `src/components/ui/Navbar`: navbar-specific components and hook
- `src/components/ui/Accordion`: accordion component + styles

## Route Overview

- `/`: intro/landing page
- `/about`: profile summary
- `/certifications`: certification accordion
- `/contact`: contact card
- `/clinicalskills`: clinical skills placeholder
- `/education`: education placeholder
- `/services`: services placeholder
